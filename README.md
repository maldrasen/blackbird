
# Blackbird
This is one of those long running, off again, on again, game development projects that I think every software engineer
is required to have. This game is something I've been working on in various forms for many years, and this is probably
at least the fourteenth time completely redoing everything from scratch. Who knows, maybe this time I'll be able to
create something that other people might actually be able to play.

### An Era Type Game
The Era Games (https://wiki.eragames.rip/index.php/Main_Page) are a fairly obscure subgenre of erotic "training" type
games popular in Japan. Though people are still making heroic efforts to translate a few of the games into Engligh,
the original source is written in a BASIC like language and is mostly completely insane.

The Blackbird project is an attempt to create a game with similar mechanics in a different insane language
(JavaScript), chosen because the game is mostly text based, and Electron just kind of works out of the box for a cross
platform game with an HTML frontend.

I'm still in the 'early' stages of development, and I doubt I'll have anything remotely playable for at least a year,
but every open source project needs a Readme just in case people somehow find this.
