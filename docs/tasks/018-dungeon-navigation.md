We have the dungeon being built, now we need to party to navigate the dungeon. The dungeon should start with all the rooms hidden. Moving to a new room reveals it on the map. The current room should be highlighted. We need to click on doors to enter new rooms. Doors are probably too small at the moment so tile size should be increased. Clicking on other rooms should move to that room, one room at a time. Now that we have battles we can also randomly start battles as we navigate the dungeon.
