I started thinking about this when doing the persisted actions and some of the action dialog. When a character is unwilling to do an action, we should look at their personality archetype's denial style to see what they do. A violent character will try to fight the player off, preventing the action entirely. Some characters react angrily making hate marks easier. Some could even get aroused more when they're forced against their will into an action. I think I want to get the Marks working before starting this, and probably at least a bare bones combat system to determing if people are able to put up a fight against the player.
