Now that we have at least a working version of the game's primary systems we can start work on the actual new game event. I have an idea of the initial scenario (the lonely shepard) that is played when the game is first started. Additional scenarios are unlocked through game play, but for the first run you're limited to being a human male, the least interesting thing to be.

### Character Creation
We should start with character creation right away. Character creation will also serve to tell the character's backstory. We'll go though a couple of quick stories from the character's past and ask how they responded to a particular situation or another. Their responses add to the character's initial stats. So if you reacted to getting bullied by beating someone up, you'll get points in strength, but maybe get points subtracted from beauty. We do maybe three of these stories. The stories are doin double duty, letting us explain the character's origin in the farmlands around Wolgur without reading too much like a block of exposition. 

##### Important details to cover here.
- Your father was an adventurer who disappeared in the dungeon shortly before you were born. (Which implies that your younger sisters and brother have a different father.)
- Your older brother set out to become an adventurer himself, but hasn't been heard from since. 
- Your family own a home in Wolgur, but left it empty after leaving the city for the farms.
- It sat empty for 15 years. Your brother may be living there now. 
- You may or may not have fucked sheep. Everyone seems to assume you have though.

During character creation, in the last story is used to select an aspect. This story should detail you traveling to the city, finding your family home in ruins, no sign of your brother except for a makeshift camp in the home's living room. 

We also need to name the character at some point. Perhaps when you get to Wolgur and the guard asks for your name.

### Starting Equipment
Once this event is done you should be at the home / living room location. You'll have some default starting equipment. The equipment should determined by your attributes, which were set during the scenario. Starting the game with high strength, dexterity, or vitality will probably be the norm. Starting as intelligent or beautiful character is tricky. You can't start the game with magic as magic has to be granted by the dungeon. Beauty is useful for gaining and  controlling your party members, but you won't have any of them at first either. You'd probably start the game with some light leather armor and some easy to use weapon like an axe and hope for the best I guess.

### Accepting Death
Even though this is a sex game, it's also a fairly unforgiving dungeon crawler. You're not allowed to save in the dungeon and death is always possible. So, because death is frequent, I'll include a type of rogue-lite meta progression into the game as well, almost to reward a character's death.

You can always reload a same game file if you die in the dungeon, but I want to add a mechanic that lets you accept the character's death. Accepting a character's death can unlock new scenarios, giving you different options at character creation. When you accept death, the last character is marked as dead and we start a new game with whatever new scenario you may have unlocked. (Could be the same as the last one if you haven't progressed at all.)

Some content will necessarily be gated behind this accept death mechanic, the ability to start the game as a woman or another species for instance.

### Creating a Lineage
This kind of progression can usually only happen once, so before the first scenario is even started we create a lineage. A lineage has one currently alive character, but keeps track of all the previous character's progress and decisions. The game's save file is actually the lineage as even when a character is killed and we start a new game, the data in the lineage is the same. In some of the planned content, Lief's story for instance, Lief can join your household or set himself up independently. Whatever his situation is will be persistent across the lineage. 

As you play the game you fix and upgrade the player's family home. You build rooms, workshops, characters (who act as the game's crafting systems) will move into the home. These upgrades are part of the lineage. And new characters can begin the game with the upgrades from the previous game. 

Eventually we'll unlock more scenarios which allow you to play the other species in the game, allowing characters to start with some magic, or party members, people already living in the player's house. All that.

##### Second Scenario
After you accept the first character's death, the second scenario will give you the option to play as the first character's younger brother or sister. The character creation will be similar but with slightly different options, and an expanded options for aspects.
