Before we can start work on the character starting equipment we need to go back and complete all the different types of base equipment. First, some of the weapon types are completely empty, with comments for what base weapons to make for those types. 

Every base armor needs to have a base damage reduction for the physical damage types: (crush, slash, pierce). Magic damage resistance is something we'll have to figure out later. It would make sense that armor would protect against fire damage, but not against the blind effect which uses the character's fire resistance to avoid. 

Every base weapon and armor will also need a value. Value partially mechanical. A base armor with higher resistances should be more valuable. Value is also based on how the armor was constructed. There's a big price hike between the leather and the metal armors. Armor that use more materials or labor are more expensive.
