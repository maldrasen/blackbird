The capture mechanics need a complete redesign. Capturing a monster is a second path available to recruit monsters, separate from the negotiation system. Captured monsters start out uncontrolled and unhappy, requiring a lot of work before they can be included in the party. Some monsters though can't be negotiated with, so this is the only option for them. 

Capturing a monster should be something that happens in battle. This can be done with the application of status effects: bound arms, bound legs, fully bound. Even if these status effects are applied magically, they'll need rope or manacles or something for when the magic wears off. 

A player should only be allowed to have a set number of captured monsters at a time. Perhaps one at first, and that could be extended later with an unlocked aspect. This capture option should only be allowed once the player has expanded their home with a prison of some sort.
