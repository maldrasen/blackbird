When a character is created with a trigger like 'strong' they'll get extra strength at character creation. These triggers should also be turned into aspects so that as a character levels they get bonus points to that attribute when leveling. They should still have to select that attribute to raise for the bonus to be applied. (The logic here is that an 'ugly' character wouldn't lose beauty every level, they would just get less than they should if that attribute is raised.)

If they get these aspects after character creation I suppose it needs to be applied retroactively? Maybe it's only half as effective though?
