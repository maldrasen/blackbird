There are really two parts to this task. The first is easy, rewriting the game over text to explain the two options. "You can try again or move on. Accepting your death will advance the story." We should only show this option though if there's actually a new scenario that's been unlocked. I think it's too punishing to have a player accept death and have to start the same scenario over with no progress to show for it.

The second part is more involved as in order to accept death, we need to add a new starting scenario, this time from the player's younger brother or sister's perspective. Same kind of setup this time, a few questions, choose your gender and get a better starting aspect this time. 

The scenario will need to define it's unlock conditions. Could be as simple as advancing to level 5 first, or could be something more specific.
