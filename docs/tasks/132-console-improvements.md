I'd like to add a few improvements to the app console. First, I've disable text selection globally in the app, because this is a game and not a document, text selection interferes with drag and drop and such. In the console though I do want to be able to cut and paste error messages and entity IDs.

Also I want to add a function to print the character roster. Finding an entity id is tricky sometimes, so having a table with character names and ids would make that easier. We can make the entity IDs clickable too, to print the component data for that entity.

It might be handy to have a way to dump the state objects to the console as well.
